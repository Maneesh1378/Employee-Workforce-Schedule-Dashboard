# Employee Workforce Schedule Dashboard

This project demonstrates an Excel-based Employee Workforce Schedule Dashboard used to track employee work schedules and analyze workforce distribution.

## Project Overview
The dashboard visualizes employee working days, sick leaves, holidays, and weekoffs for a monthly workforce schedule.

## Key Features
- Total Employees KPI
- Days Worked Analysis
- Sick Leave Tracking
- Holidays Monitoring
- Weekoff Tracking
- Employee Work Distribution Charts

## Key Insights
- James worked the highest number of days (23 days)
- Singh had the highest holidays (6 days)
- Danny had the highest weekoffs (5 days)
- Total working days in January: 65
